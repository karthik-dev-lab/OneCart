# 🛒 OneCart — AI Voice-Controlled E-Commerce Platform

🔗 **Live Demo:** https://onecart-ecommerce-1y32.onrender.com/  
🔗 **GitHub Repo:** https://github.com/karthik-dev-lab/OneCart  

---

## 🚀 Overview
OneCart is a full-stack MERN e-commerce platform enhanced with **AI-powered voice navigation**, allowing users to browse, search, and interact with the application using voice commands.

---

## ✨ Key Features

- 🛍️ Full-featured e-commerce system (products, cart, orders)
- 🔐 Google Authentication using Firebase
- 💳 Secure payments via Razorpay
- 🧠 AI Voice Navigation (Speech Recognition + Text-to-Speech)
- 🛠️ Admin Dashboard (Manage products, users, orders)
- 📱 Fully responsive UI (mobile + desktop)
- ⚡ Real-time navigation and state updates

---

## 🎤 AI Voice Navigation (Core Highlight)

Users can control the app using voice commands like:
- “Open search”
- “Go to cart”
- “Show products”
- “Open orders”

### Tech Used:
- Web Speech API (SpeechRecognition)
- SpeechSynthesis API (Voice feedback)
- React Router (dynamic navigation)

---

## 🧰 Tech Stack

**Frontend:**  
- React.js  
- Tailwind CSS  

**Backend:**  
- Node.js  
- Express.js  

**Database:**  
- MongoDB  

**Integrations:**  
- Firebase (Google Auth)
